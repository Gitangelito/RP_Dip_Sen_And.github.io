# 🇵🇪 RP Lima 2026 — Guía de Voto Informado

Dashboard interactivo de análisis de candidatos de **Renovación Popular** para las **Elecciones Generales 2026** en Lima, Perú.

---

## ⚠️ Disclaimer importante

Este proyecto es **100% educativo e informativo**. No es un proyecto profesional, periodístico ni afiliado a ningún partido político, institución electoral ni organización. El análisis fue construido con información pública disponible en fuentes verificables como el JNE, La República, Infobae Perú, El Comercio, RPP, Wikipedia y la Procuraduría.

**Las clasificaciones de riesgo son opiniones basadas en evidencia pública — no son sentencias ni dictámenes legales.**

Si encuentras un error o tienes información adicional verificable, abre un issue o un pull request.

---

## 📋 ¿Qué incluye?

- **30 senadores** — Distrito Único Nacional
- **Senadores** — Distrito Múltiple (Callao, Lima)
- **32 diputados** — Lima Metropolitana
- **14 candidatos** — Parlamento Andino
- Filtros por cámara y nivel de riesgo
- Recomendaciones basadas en trayectoria, ética e independencia
- Completamente mobile-friendly

---

## 🔍 Metodología

Cada candidato fue evaluado en 4 dimensiones, del 1 al 10:

| Dimensión | Descripción |
|---|---|
| **Integridad** | Historial limpio, ausencia de condenas o investigaciones graves |
| **Competencia** | Formación académica verificable y experiencia real en gestión pública |
| **Riesgo de red** | Vínculos con actores corruptos, conflictos de interés, redes de influencia cuestionables |
| **Ética** | Conducta pública, coherencia en declaraciones, transparencia |

### Niveles de riesgo

| Nivel | Significado |
|---|---|
| 🔴 Alto riesgo | Señales graves documentadas: condenas, investigaciones activas, conflictos de interés probados |
| 🟡 Riesgo medio | Señales de alerta moderadas: oportunismo político, información limitada, vínculos cuestionables |
| 🟢 Bajo riesgo | Perfil limpio con trayectoria verificable y sin antecedentes graves |
| ⬜ Sin datos | Información pública insuficiente para evaluación — no implica culpa ni inocencia |

**Importante:** "Sin datos" no significa malo. Significa que no hay suficiente información pública disponible para evaluarlo. La mayoría de candidatos en posiciones altas de la lista de diputados y todo el Parlamento Andino caen en esta categoría — eso es una realidad del proceso electoral peruano, no un juicio sobre las personas.

## 🚀 Cómo usar

Es un archivo HTML estático. No necesita servidor ni instalación:

```bash
# Clona el repositorio
git clone https://github.com/tu-usuario/rp-lima-2026.git

# Abre el archivo en tu navegador
open RP_Lima_2026.html
```

O simplemente descarga el archivo `RP_Lima_2026.html` y ábrelo en cualquier navegador — Chrome, Safari, Firefox. Funciona en celular y computadora.
